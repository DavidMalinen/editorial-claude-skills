# Spread Archetype Ideation

Apply four canonical editorial archetypes — each with its own lineage, governing rules, and tests — to design spreads that emerge from content rather than decoration.

The four archetypes:

1. **Data spread** — quantitative information made visually honest (Isotype → Swiss → Tufte)
2. **Ratio/specimen spread** — proportion and typographic form as aesthetic objects (Bodoni → Tschichold → Weingart → Bill)
3. **Fragmented Merz spread** — typography as autonomous compositional element (Schwitters → Dada → Lissitzky → Bauhaus)
4. **Multi-column prose spread** — sustained reading at industrial scale (medieval canon → Tschichold → Müller-Brockmann → Bringhurst)

## Workflow

### For new spreads:

1. **Clarify what the content wants to do**
   - Is it primarily quantitative, typographic, poetic, or narrative?
   - Is the reader scanning, contemplating, deciphering, or reading at length?
   - What relationship between text and image does the content imply?

2. **Select the archetype** using content type as the primary signal:
   - **Quantitative / comparative** → Data spread
   - **Display of proportion, character, or specimen** → Ratio/specimen spread
   - **Poetic, manifesto, fragment, sound-poem, declaration** → Fragmented Merz spread
   - **Sustained prose with figure interruptions** → Multi-column prose spread

   The selection is not aesthetic preference. A long argumentative essay placed in a Merz spread is unreadable; a data report placed in a prose grid hides its argument. Content type drives form.

3. **Apply the archetype's governing rules** (see `references/four-archetypes.md`)
   - Each archetype has its own canon of rules. Do not mix freely. A spread that violates its own archetype reads as decorative chaos; a spread that violates the *wrong* archetype's rules reads as confused.

4. **Apply the archetype's whitespace logic**
   - Data: whitespace can encode absence, zero, baseline
   - Specimen: whitespace is structural silence between typographic events
   - Merz: whitespace is tension, eye-direction, the silence between sounds
   - Prose: whitespace is margin geometry (Van de Graaf 2:3:4:6) and breathing rhythm

   Whitespace is never residual. If you cannot say what a given empty area is *doing*, it is not working.

5. **Apply the integration test before shipping**
   - Are words, numbers, and images interleaved, or are they segregated into zones?
   - For data: do labels sit on the data, or in a legend?
   - For specimen: does the numeric/typographic scale appear in the same eye-frame as the form being demonstrated?
   - For Merz: is text functioning compositionally (as visual mass) as well as linguistically?
   - For prose: do figures appear near their first text reference, on column grids, at column tops/bottoms?

6. **Apply the archetype test** (see `references/four-archetypes.md` for each)

### For critiquing spreads:

1. **Name the archetype the spread is trying to be**
   - If you cannot name it, the spread is either confused or it is a genuine hybrid (rare — usually confused).
   - If the spread mixes archetypes, identify the dominant one and check whether the secondary elements undermine or support it.

2. **Check against that archetype's failure modes** (see archetype-specific failure lists in `references/four-archetypes.md`)
   - Data spread failures: scaling by diameter not area, missing labels, decorative color, Lie Factor > 1.05
   - Specimen failures: arbitrary type sizes (no scale), centered timidity, weak proportional logic
   - Merz failures: chaos without underlying structure, decorative angles, fragmentation that doesn't slow reading
   - Prose failures: lines too long (>75 char) or too short (<40 char), figures mid-column, captions same size as body, weak margin proportions

3. **Check whitespace function**
   - For every empty region: what is it doing? If "nothing" is the answer, the layout is residual, not active.

4. **Check the lineage**
   - Does the spread honor or violate the tradition it descends from? Violation is fine if intentional and earned; accidental violation is the most common failure.

5. **Suggest improvements** with specific before/after recommendations naming the archetype rule being applied.

## Quick checklist

- [ ] Archetype identified and content type matches
- [ ] Governing rules for that archetype satisfied
- [ ] Whitespace is doing identifiable work
- [ ] Words, numbers, images integrated within their eye-frame
- [ ] No covert mixing of archetypes
- [ ] Proportional honesty (Lie Factor ≈ 1.0 if quantitative)
- [ ] Typographic scale present, not arbitrary
- [ ] Figures placed by grid logic, not eyeballing
- [ ] Underlying structure visible even where surface looks chaotic (Merz)
- [ ] Reading rhythm appropriate to content (slowed for poetry, sustained for prose)
